DashBoard - Prefeitura

Este aplicativo simula o consumo de uma API disponibilizada pela prefeitura do recife, e appresenta o resumo das informações!

Sistema Web desenvolvido em JAVA, utilizando JSF + Primefaces, Bootstrap e JQuery. Em resumo, ele acessa a API da prefeitura onde possui mais 1Mi de registros de informações financeiras e agrupa em graficos e tabelas totalizadoras.,
